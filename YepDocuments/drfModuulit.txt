DRF-projektissa yleisesti käytetyt moduulit
==========================================

1. Automaattisesti käytettävissä olevat (ei tarvitse asentaa erikseen)
----------------------------------------------------------------------
- os                 | Käyttöjärjestelmän kanssa toimiminen (esim. tiedostopolut)
- datetime           | Päivämäärien ja aikojen käsittely
- json               | JSON-tiedon lukeminen ja kirjoittaminen
- re                 | Säännölliset lausekkeet (regex)
- uuid               | Yksilöllisten tunnisteiden luonti
- math               | Matemaattiset toiminnot
- logging            | Lokitusjärjestelmä
- pathlib            | Polkujen käsittely objektimaisesti

2. Oletuksena DRF-projektissa mukana (jos DRF on asennettu)
-----------------------------------------------------------
- django.http                      | HTTP-pyynnöt ja -vastaukset
- django.urls                      | URL-reititys
- django.shortcuts                 | Avustajat kuten get_object_or_404
- django.db                        | Tietokantatoiminnot
- rest_framework.views             | API-näkymät
- rest_framework.response          | Response-objekti
- rest_framework.serializers       | Datan sarjoittaminen
- rest_framework.viewsets          | CRUD-toiminnallisuus
- rest_framework.permissions       | Käyttöoikeuksien hallinta

3. Ulkoiset moduulit – pitää asentaa erikseen
---------------------------------------------
- requests              | pip install requests          | HTTP-pyynnöt ulkoisiin palveluihin
- httpx                 | pip install httpx             | Moderni vaihtoehto requestsille
- pytest                | pip install pytest            | Yksikkötestaus
- coverage              | pip install coverage          | Testikattavuuden mittaus
- pillow                | pip install pillow            | Kuvankäsittely
- django-cors-headers   | pip install django-cors-headers | CORS-ongelmien ratkaisu
- drf-yasg              | pip install drf-yasg          | Swagger-dokumentaatio
- python-dotenv         | pip install python-dotenv     | .env-tiedostojen käsittely
